#Evaluate labs TASK 1

 Scan Your Local Network for Open Ports

 Steps and procedure :
 
 Step 1: install nmap tool from it's official website
 step 2: now find your ip by typing this 'ipconfig' on command prompt
 step 3: now type this command on in the nmap tool  nmap -sS 192.168.1.0/24 to perform TCP SYN scan.
 step 4: now click the scan button and now you can see the open ports 
 step 5: now we can see that netbus is accesing a open port 12345 in TCP protocol, which can be risk to our system
 step 6: now we can block that port in Windows Defender Firewall with Advanced Security ![Screenshot (182)](https://github.com/user-attachments/assets/b244581f-d946-4e00-bd17-cd1c881cb190)
